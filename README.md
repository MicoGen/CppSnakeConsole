## CppSnakeConsole
## v1.0.0 Beta release:
#### Functions:

1. Setup: initialise game variables as game state, score, pointer direction, places of the game objects;

2. Draw: draw playing fielg, game objects as fruits and snake (head and body), player score;

3. Input: observe user press direction buttons as W, S, A, D; and exite button as X;

4. Logic: change places of the game objects depending of direction variable, and process several game situation as
snake collide fruit, game field or her body.

#### Features:
* Snake not die when collide wall or herself (instead she stop maving).
